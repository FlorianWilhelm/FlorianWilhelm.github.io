How to Setup a new Python Project

In this presentation held at the EuroPython 2014 in Berlin, my colleague Felix Wick demonstrates that setting up a new Python project from scratch can be quite complicated. Especially, structuring your files and directories the right way in order to build a Python package. Thereby, he addresses questions like “Where should my modules, documentation and unit tests go?” and “How do I configure setup.py, Sphinx and so on?” with proven answers! Finally, he demonstrates the template tool PyScaffold which helps a lot in setting up a new project.
