We also drew inspiration from real-world <span class="caps">AI</span> and Streamlit applications, as we wanted to combine common components into an easy-to-understand application that could be used for learning purposes by exploring the source&nbsp;code.</p> <p>Our application crawls the Snowflake documentation at <a href="https://docs.snowflake.com/en/">https://docs.snowflake.com/en/</a> and saves it in a Snowflake table. The actual Snow Instructor app is now a <a href="https://streamlit.io/">Streamlit</a> app that reads a page from the stored Snowflake documentation and passes it to the Artic <span class="caps">LLM</span> to generate a question. This question is now presented to the user as a quiz with four possible answers to choose from.
